webpack 入门

一、什么是webpack

webpack是近期最火的一款模块加载器兼打包工具，支持commonJS和AMD模块。 支持很多模块加载器的调用，可以使模块加载器灵活定制，比如babel-loader加载器，该加载器能使我们使用ES6的语法来编写代码。 可以通过配置打包成多个文件，有效的利用浏览器的缓存功能提升性能。 使用模块加载器，可以支持sass，less等处理器进行打包且支持静态资源样式及图片进行打包。

二、安装和配置

安装

-在项目里安装 ：npm install webpack --save-dev
-全局安装 ：npm install webpack -g

三、运行

运行webpack
$ webpack --display-error-details
后面的参数--display-error-details是推荐加上的，方便出错时能查阅更详尽的信息（比如 webpack 寻找模块的过程），从而更好定位到问题。

其他参数

$ webpack --config XXX.js //使用另一份配置文件（比如webpack.config2.js）来打包
$ webpack --watch //监听变动并自动打包
$ webpack -p //压缩混淆脚本
$ webpack -d //生成map映射文件，告知哪些模块被最终打包到哪里了
